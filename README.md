# Newsify-Real-Time-Narrative-Intelligence-Dashboard

🔹 Business Problem

Financial analysts face information overload from real-time news streams.

🔹 Solution Overview

Developed an interactive dashboard that ingests live news, performs sentiment analysis, extracts entities, and generates AI-powered executive briefings.

🔹 Architecture

Amplyfi API
↓
Data Processing (Pandas)
↓
NLP (spaCy + VADER)
↓
Sentiment & Entity Extraction
↓
Streamlit Dashboard
↓
AI Summary (Gemini API)

🔹 Tech Stack

Python

Streamlit

Pandas

spaCy

NLTK (VADER)

Google Gemini API

Plotly

🔹 Key Implementation

Real-time news ingestion

Named Entity Recognition

Sentiment trend visualisation

Interactive filtering

AI-powered summary generation

🔹 Measurable Impact

Enabled rapid narrative risk analysis

Reduced research time

Delivered 5-second executive summaries

🔹 Enterprise Value

Supports proactive decision-making

Improves intelligence gathering

Enhances financial risk monitoring
